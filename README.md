A custom, high-performance multiplayer mod for Geometry Dash, built on top of the **Geode** framework. This mod acts as an open-source alternative to Globed, specifically tailored and optimized to run on **GDPS (Geometry Dash Private Servers)**.

Experience Geometry Dash with your friends on your favorite private servers, featuring real-time position syncing, custom cosmetics, and seamless server-side integration.

---

## 🚀 Features

- **GDPS Compatibility:** Fully compatible with custom Geometry Dash Private Servers. Easily point the mod to your GDPS backend.
- **Real-time Multiplayer:** High-tickrate position, animation, and state synchronization with minimal latency.
- **Geode Native Integration:** Built using the Geode SDK for maximum performance, stability, and compatibility with other Geode mods.
- **Room System:** Create public or password-protected private rooms to play with specific groups of friends.
- **Customization:** Support for icons, ships, waves, and custom server-defined cosmetics.
- **Lightweight Protocol:** Uses an optimized networking protocol (UDP/WebSockets) to ensure smooth gameplay even on low-end servers.
